About

This training is brought to you by codecentric AG. We're always looking for talented devs who share our passion for software craftsmanship. If you want to become part of an awesome team, visit our website or talk directly to your trainer.

If you like the worblehat demo application, you can follow it's development at github.

The development infrastructure is setup using docker containers, which are orchestrated via fig. It is running on medium EC2 instances at AWS. You can try it at home buy following the instructions in the github repository.
